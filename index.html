Como Turbinar Qualquer Prompt
2 upgrades que eliminam respostas genéricas — clique para expandir

Upgrade 01
Especifique o Público-Alvo
Todo prompt com "público geral" produz conteúdo genérico. Adicione este bloco no início de qualquer prompt:
PÚBLICO-ALVO: [seja específico] FAIXA ETÁRIA: [ex: 28–42 anos] CONTEXTO DE VIDA: [ex: mães que trabalham, executivos de TI] DOR PRINCIPAL: [ex: ansiedade pós-trabalho] NÍVEL COM O TEMA: [nunca tentou / já desistiu / intermediário]
Exemplos por nicho:
🍽️ Receitas → "mães com menos de 30 min por refeição"
🧘 Meditação → "executivas com ansiedade pós-reuniões"
💰 Vendas → "freelancers que nunca venderam por DM"
📦 Dropshipping → "universitários com R$500 de capital"
📱 Instagram → "negócios locais com menos de 200 seguidores"
Upgrade 02
Exija Variação de Cenário
Sem instrução, a IA repete os mesmos cenários (praia, floresta, café). Para forçar variedade, adicione:
VARIAÇÃO OBRIGATÓRIA: - NÃO repita clichês como [ex: praia/floresta] - Gere 3 VERSÕES DISTINTAS: A: [contexto urbano/profissional] B: [cotidiano doméstico/familiar] C: [abstrato/simbólico/emocional] - Cada versão: ative 3+ sentidos diferentes
Aplicações práticas:
🧘 Meditação → não só praia: biblioteca com chuva, bolha de luz, quarto de infância
✨ Afirmações → não só "abundância": relacional, criativa, saúde
🍽️ Receitas → não só família: jantar a dois, almoço solo, reunião de amigos
💰 Vendas → não só "dor genérica": 3 situações concretas do avatar
